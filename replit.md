# Static HTML Website

## Overview
This is a static HTML website - a collection of HTML pages that appear to be from replit.com. The site is served using a simple Python HTTP server.

## Recent Changes
- 2025-11-16: Initial setup - configured Python HTTP server to serve static HTML files on port 5000

## User Preferences
- Language: Arabic (العربية)

## Project Architecture
- **Type**: Static HTML website
- **Server**: Python HTTP server (server.py)
- **Port**: 5000 (frontend)
- **Files**: Collection of HTML pages in various directories (gallery/, products/, customers/, etc.)
